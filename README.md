# Tosin4dev — The OS I need for dev

Local-first AI work-orchestration console. Multi-board Kanban where approved tickets run autonomously through Claude Code / Codex CLI / Hermes runners to a review-ready draft PR. Human gates: spec approval + final review.

## Stack (planned)

- TanStack Start (front + back, no separate API)
- TanStack Query via react-query-kit
- MongoDB (local container)
- Bun
- Visual language adapted from gray-ui-csm
